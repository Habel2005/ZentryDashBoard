# **App Name**: Zentry Admin UI

## Core Features:

- Dashboard Overview: Display key performance indicators (KPIs) such as total users, total sessions, active sessions, and completed sessions using mock data.
- Sessions List: Browse and filter sessions with pagination, a search bar, and a status filter. Allows drilling down into specific sessions.
- Session Detail View: Show session transcripts, summaries (with 'tool' indication if generative AI is applicable), and seat management options.
- Transcript Viewer: Display session messages in a clean, message bubble UI, showing a tool callout in the AI powered agent messages
- Seat Management: Manage seat availability, including updating seat counts via an editable modal.
- User Management: List and manage users with basic details, including the ability to view filtered sessions.
- Admin Settings: Configure mock admin-specific settings such as retention period, cache TTL, and summary enabling.

## Style Guidelines:

- Primary color: #3F51B5 (Deep Indigo) for a professional and trustworthy feel.
- Background color: #F5F5F5 (Light Gray) for a clean, minimalist look.
- Accent color: #FFAB40 (Soft Orange) for interactive elements and highlights.
- Font: 'Inter' sans-serif for body and headline; clean and readable.
- Icons: Material Icons or FontAwesome (consistent across app).
- Clean cards, subtle shadows, rounded corners, smooth transitions using Tailwind CSS.
- Gentle micro-animations for improved user experience.
